# STEPPSフレームワーク（口コミが自然発生する構造）

## 🗓 日付
2026-05-03

## 参考URL
- *Contagious* by Jonah Berger（書籍）

## 💡 学んだこと（要約）
### 1. 何を知ったか（一言）
「口コミが起きる構造があるか」を6要素で評価するマーケティングフレームワーク。

### 2. 以前の知識とどう繋がったか（もしあれば）
「集客を設計するツール」と誤解しやすいが、STEPPSが問うのはあくまで口コミ発生の構造的条件。
集客はその副産物であり、直接コントロールするものではない。

## 使い方・要素一覧
| 要素 | 内容 |
|------|------|
| Social Currency | 「これ知ってる自分かっこいい」と思わせる |
| Triggers | 日常の何かが製品・アイデアを思い出させる |
| Emotion | 強い感情（驚き・怒り・感動）が共有を促す |
| Public | 他者に見えるほど模倣・拡散されやすい |
| Practical Value | 役立つ情報は「誰かに教えたい」を生む |
| Stories | 物語に乗せた情報は記憶に残り再発信される |

事前設計にも事後分析にも使える。「この施策で口コミが起きる構造があるか？」をチェックするフレームワーク。
