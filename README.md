## TIL(Today I Learned)
#### 概要
日々の勉強を雑に書き留めていく場所です。<br>
完成度は気にせず、未来の自分が「あれどうだったっけ」と思ったときに引っかかればOKというスタンスで書いています。

#### リポジトリの見方<br>
til/<br>
├── android/<br>
├── Kotlin/<br>
└── res/
<br>

- Android：JetpackCompose メイン
- Kotlin：JetpackCompose に使われる構文など
- res：画像など
